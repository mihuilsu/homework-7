Box-sizing Property Examples

box-sizing: content-box (default)

Width and height include only the content. Padding and border are added outside.

Content-box
Width: 200px + Padding: 2×20px + Border: 2×5px = Total: 250px

box-sizing: border-box

Width and height include content, padding, and border.

Border-box
Total width stays 200px (content area shrinks)
